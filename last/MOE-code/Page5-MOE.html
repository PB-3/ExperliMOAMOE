Liste des risques :

Vous devez remplir les informations concernant les risques identifiés. Pour chaque risque, évaluez sa gravité sur une échelle de 1 à 5 et estimez les chances d'apparition sur une échelle de 1 à 5.

Ces évaluations vous aideront à établir une analyse de risque pour votre projet.

Risque : Pertes de données lors de la migration vers une nouvelle plateforme.

Risque : Problèmes de compatibilité avec différents navigateurs web.

Risque : Vulnérabilités de sécurité non corrigées dans le site existant.

Risque : Retard dans la livraison du projet en raison de contraintes techniques imprévues.

Risque : Insatisfaction du client avec les modifications apportées au site.
